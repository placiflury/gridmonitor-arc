Error: GIIS List Missing in Deployed Service Configuration

The service started with a configuration that did not specify any Grid Index Information System (GIIS) servers to be querried. The GIIS server(s) provide a 'bootstrapping' of the monitoring system and are therefore crucial.

Remedy

Open your configuration file (most likely called YOUR_APPLICATION_PATH/GridMonitor/production.ini and enter the following lines in the [app:main] section:

giis = giis_server_1[:port]

or if you have several giis_servers:

giis = giis_server_1[:port] , giis_server_2[:port] , ...
